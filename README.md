# MilkersHub

GitHub-only hub for the idea behind [@milkers](https://x.com/milkers):
convert social attention into transparent community updates and measurable on-chain actions.

> Scope of this repo: **only GitHub infrastructure** (docs, updates, community workflow).  
> No website/app in this version.

## Links
- X: https://x.com/milkers
- Token (Pump): `nNcqotFpBF7hsAqbNs1v2cYWeMnaKjFc3D2gHoQpump`

## Core principles
1. **Social momentum** — community growth and visibility.
2. **Transparency** — public updates and changelog.
3. **Execution** — clear actions, owners, and dates.

## Workflow
1. Open an issue (idea / task / bug).
2. Link it to a PR.
3. Merge with a short changelog note.
4. Add public update in `docs/updates/`.
