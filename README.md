GitHub-only coordination hub for the idea behind [@milkers](https://x.com/milkers).

## What’s inside
- Public roadmap and weekly updates
- Issue/PR templates for clean execution
- Contributor rules and code of conduct
- Markdown CI check via GitHub Actions

## Quick workflow
1. Open issue (`Feature / Task / Bug`)
2. Submit PR linked to issue
3. Merge and add an update in `docs/updates/`
